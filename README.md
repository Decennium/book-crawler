# Book Crawler(网络小说下载器)

base on Celery & Tornado

# 安装

* 修改 celeryconfig.py， 参考 http://www.celeryproject.org/
* 运行 celeryd
* 运行 python website.py
* 可以编辑 rules.py 添加网站规则已支持更多小说网站
